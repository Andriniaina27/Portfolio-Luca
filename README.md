# Portfolio Angular 17

Portfolio de RANAIVO Andriniaina Luca, migré depuis HTML/CSS/JS vers Angular 17 (standalone components).

## Lancer le projet

```bash
npm install
npm start
```

Puis ouvrir [http://localhost:4200](http://localhost:4200).

## Build production

```bash
npm run build
```

## Structure

- `src/app/components/` — header, hero, projets, compétences, parcours, contact, footer, particules
- `src/app/directives/reveal.directive.ts` — animations au scroll
- `src/assets/dossier/` — images et CV
- `_legacy/` — version HTML/CSS/JS d'origine
